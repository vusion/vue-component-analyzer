## 安装

``` shell
npm install --save-dev vue-component-analyzer
```

## 选项

#### name

某个功能的名称。

- Type: `string`
- Default: `'hello'`

#### auto

是否自动做某件事。

- Type: `boolean`
- Default: `'true'`
